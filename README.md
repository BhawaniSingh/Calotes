Calotes
====
All the code is licensed Under GNU GPL v3

An open source application to add license to your source code.

Currently supported languages are:

	1. Java
	2. C/C++
	3. Python
	4. Objective-c
	5. Javascript
	
Currently supports following file extensions

	18. C
